# ALEEM CAR SALES Website

A professional car dealership website with WordPress-style admin dashboard.

## Features

- **Main Website**: Car inventory, services, financing options, contact information
- **Admin Dashboard**: Manage cars, services, and financing options
- **Responsive Design**: Works on all devices
- **Local Storage**: Data persists between sessions

## Quick Start

1. Open `index.html` in your browser to view the main website
2. Click "Admin Dashboard" to access the admin panel
3. Add cars, manage services, and update financing options

## Admin Features

- Add/edit/delete cars
- Mark cars as available or sold
- Manage services and financing options
- View dashboard statistics

## Deployment

This site can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Technology

- Pure HTML, CSS, JavaScript
- localStorage for data persistence
- Responsive CSS Grid and Flexbox
- Font Awesome icons
